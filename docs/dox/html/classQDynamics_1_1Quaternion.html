Detailed Description

A computational replica of the mathematical object quaternion, which replicates the expected behaviour of such objects, including addition and scalar multiplication (which it inherits from

embed:rst:inline `JSL::Vector() <https://jackstandardlibrary.readthedocs.io/en/latest/vectors.html>`_

), as well as the idiosyncratic multiplication and division native ot this field. We use the notation that

embed:rst:inline :math:`\mathsf{q} = (q_0, \vec{q})`

, and the interface is based on this assumption. It is intentional that treating quaternions as members of

embed:rst:inline :math:`\mathbb{R}^4`

is awkward.
